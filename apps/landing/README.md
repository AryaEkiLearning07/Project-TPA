# Landing App

Folder untuk landing page publik (`tparumahceria.my.id` / `tparumahceria.com`).

## Menjalankan Lokal

- Landing page: `npm run dev:landing` lalu buka `http://localhost:5174`
- Platform utama (admin/petugas): `npm run dev` lalu buka `http://localhost:5173`

Keduanya bisa jalan bersamaan karena menggunakan port berbeda.
